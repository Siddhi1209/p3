Autosys  - To automate the workflow . 

Automate - [Event base / Time base ]

1. Event base  - > Automation tool 

2. Time base -> Either Automation tool or  CRONTAB 

Components  - 
1. Event Server
2. Scheduler 
3. Agent 


OS compatibility  - 

1. Windows
2. HP-UX
3. Linux

Supported Database - 

1. Microsoft SQL Server
2. Oracle 
3. Sybase
4. Postgre [Dec 2024]

Autosys Product - 
1. WAAE [Workload Automation Autosys Edition]
2. WCC [Workload Control Centre]
3. EEM [Embeded Entitlement Manager]

Command in autosys - 
1. Jil -> Job information language
2. sendevent -> to send the event in autosys database . 
3. autorep -> To get the job detail with option [sj and qj ]
	alias sj => autorep -sj 
	alias qj => autorep -qj 

script -> It is a program [set of instructions ] that provide a desired result. 

job -> set of program. 

Job type in autosys - 

CMD
BOX  [It is a wrapper job]
--------------------------------------------------------------------------
To insert the job in autosys 
--------------------------------------------------------------------------
insert_job: <SERVER_NAME>.<owner>_<JOB_NAME>_<DQP_TYPE>
insert_job: SLAVE11.i_online_test_job   job_type: CMD
box_name: TEST_JOB
command: /usr/bin/ksh -c '$data/Autosys_scripts/auto_conn_check.sh'
machine: SLAVE11
owner: fnsonli
permission: gx
job_terminator: t
std_out_file: "$sysout/autosyslog.out"
std_err_file: "$sysout/autosyslog.err"
alarm_if_fail: t
profile: "$HOME/.profile_uajm"
alarm_if_terminated: t
send_notification: n
